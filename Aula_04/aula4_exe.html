50 Exercícios de Formulários

Instruções: Cria um ficheiro forms.html. Resolve cada exercício sequencialmente. Testa sempre no browser!

Nível 1: Estrutura e HTTP (1-5)
1 Cria a tag <form>.
2 Adiciona o atributo action="#" (para não enviar a lado nenhum por agora).
3 Define o atributo method="GET". Adiciona um input e um botão. Submete e olha para o URL do browser. O que aconteceu?
4 Muda o método para method="POST". Submete novamente. Os dados desapareceram do URL? (É assim que enviamos passwords).
5 Adiciona o atributo target="_blank" ao form. Onde abre a resposta agora?
Nível 2: Inputs Clássicos (6-15)
6 Cria um input de texto simples: type="text".
7 Adiciona um placeholder "O teu nome".
8 Cria um campo de senha: type="password". Escreve nele.
9 Cria uma caixa de seleção: type="checkbox" com o texto "Aceito os termos".
10 Faz com que a checkbox já venha marcada por defeito (checked).
11 Cria dois type="radio" (Sim/Não).
12 Dá o mesmo name="escolha" aos dois radios. Tenta selecionar ambos. (Agora só podes escolher um, certo?).
13 Cria um botão para limpar o formulário: type="reset".
14 Cria um botão de envio usando <input type="submit">.
15 Cria um botão de envio usando a tag moderna <button>Enviar</button>.
Nível 3: Labels e Organização (16-20)
16 Cria um label que diga "Nome:".
17 Dá um id="nome" ao input de texto.
18 Liga o label ao input usando for="nome". Clica na palavra "Nome". O cursor saltou para a caixa?
19 Envolve os inputs num fieldset (caixa de grupo).
20 Dá um título ao grupo usando legend (ex: "Dados Pessoais").
Nível 4: Inputs Modernos HTML5 (21-30)
21 Cria um campo type="email". Tenta enviar sem o @.
22 Cria um campo type="number". Tenta escrever letras.
23 Cria um campo type="tel" (Abre teclado numérico no telemóvel).
24 Cria um campo type="url".
25 Cria um seletor de data: type="date".
26 Cria um seletor de cor: type="color".
27 Cria um slider (barra de deslizar): type="range".
28 Cria um campo de pesquisa: type="search" (Tem o X para limpar).
29 Cria um campo para ficheiros: type="file".
30 Permite selecionar múltiplos ficheiros com multiple.
Nível 5: Dropdowns e Texto Longo (31-35)
31 Cria uma área de texto grande: textarea.
32 Define que o textarea tem 5 linhas de altura (rows="5").
33 Cria um menu dropdown: select.
34 Adiciona 3 opções (option): Azul, Vermelho, Verde.
35 Faz com que "Verde" venha selecionado por defeito (selected).
Nível 6: Validação Nativa (36-40)
36 Adiciona required ao campo Nome. Tenta enviar vazio.
37 No campo number, define min="0" e max="10".
38 No campo password, define minlength="8".
39 Adiciona readonly a um input (O utilizador vê mas não edita).
40 Adiciona disabled a um botão (Fica cinzento e não clica).
Nível 7: Ligação ao JavaScript (41-50)

Nota: Adiciona id="meuForm" ao formulário e id="msg" a um input de texto.

41 Cria uma tag <script> no final do body.
42 Cria uma função JS vazia: function validar(e) { }.
43 Liga o form à função: <form onsubmit="validar(event)">.
44 Dentro da função, escreve e.preventDefault(). (Isto impede a página de recarregar!).
45 Adiciona um alert("Enviei!"). Testa o form.
46 Vamos ler o valor do input. Cria uma variável: let texto = document.getElementById("msg").value.
47 Muda o alert para mostrar o que escreveste: alert("Escreveste: " + texto).
48 (Lógica) Adiciona um if (texto === "") para ver se está vazio.
49 Se estiver vazio, mostra alert("Erro: Escreve algo!").
50 Desafio Final: Usa console.log(texto) em vez de alert e vê o resultado na consola (F12) do browser.
